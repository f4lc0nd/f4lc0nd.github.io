Security Resources Dataset

A collection of structured security-related information

About this Project

This repository contains structured datasets related to cybersecurity resources, including tools, write-ups, events, and more. The JSON files are maintained to provide machine-readable information for developers and security researchers.

Available Datasets

events.json

Security conferences, workshops, and community events

oneliner.json

Useful command-line snippets for security tasks

pdf_info.json

Security-related PDF documents and research papers

security_tools.json

Curated list of security tools and utilities

terminal-apps.json

Command-line applications for security professionals

tools.json

General security tools and resources

write-ups.json

Collection of security write-ups and CTF solutions

Usage Example

// Fetch security tools data
fetch('https://your-username.github.io/repo-name/security_tools.json')
  .then(response => response.json())
  .then(data => console.log(data));
